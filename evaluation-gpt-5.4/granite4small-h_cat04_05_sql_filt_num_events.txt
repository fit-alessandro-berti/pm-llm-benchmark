```sql
WITH filtered_cases AS (
    SELECT case_id
    FROM event_log
    GROUP BY case_id
    HAVING COUNT(*) <= 6
)
SELECT e.*
FROM event_log e
JOIN filtered_cases fc ON e.case_id = fc.case_id;
```

This correctly:
- groups by `case_id`
- filters to cases with `COUNT(*) <= 6`
- returns all events for those remaining cases
