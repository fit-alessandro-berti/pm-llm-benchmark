### 2. Batching impact metric is partially wrong
For shipping batches, the answer proposes measuring delay as:

That only measures the delay **after the last order is ready**. It misses the fact that **earlier-ready orders may have waited much longer** for the batch to form. So it **understates true batching delay**.

A better metric would include:
- per-order batch waiting time = shipping label time  that order’s QC completion,
- plus batch assembly lag statistics,

This is a meaningful analytical flaw.
