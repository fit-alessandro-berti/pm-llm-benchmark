The core SQL is accurate:

```sql
WITH counts AS (
    SELECT case_id, COUNT(*) AS cnt
    FROM event_log
    GROUP BY case_id
)
SELECT e.*
FROM event_log e
JOIN counts c ON e.case_id = c.case_id
WHERE c.cnt <= 6;
```
