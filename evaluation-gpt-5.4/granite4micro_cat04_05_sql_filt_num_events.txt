### Better version
A more precise answer would be:

```sql
SELECT case_id, activity, timestamp
FROM (
    SELECT *,
           COUNT(*) OVER (PARTITION BY case_id) AS event_count
    FROM event_log
) t
WHERE event_count <= 6;
```

SELECT case_id, activity, timestamp
