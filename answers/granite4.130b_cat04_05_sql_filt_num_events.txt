**DuckDB SQL Query**

```sql
SELECT e.*
FROM   event_log AS e
WHERE  e.case_id IN (
    SELECT case_id
    FROM   event_log
    GROUP BY case_id
    HAVING COUNT(*) <= 6
);
```

**Explanation**

1. **Subquery (`SELECT case_id …`)**  
   - Groups the rows of `event_log` by `case_id`.  
   - Uses `HAVING COUNT(*) <= 6` to keep only those `case_id`s that have six or fewer events.

2. **Outer Query**  
   - Selects all columns (`e.*`) from `event_log` where the `case_id` appears in the list returned by the subquery.  
   - This effectively filters out any case with more than six events, while preserving every event belonging to qualifying cases (i.e., those with 6 events).
