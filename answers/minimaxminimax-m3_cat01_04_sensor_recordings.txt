## 4. Methodology Note

This is a **trace clustering** approach to process mining:
1. **Sliding-window pattern detection** identified breakpoints where two or more sensor readings changed regime simultaneously (e.g., the jump from Event 3  4 marks the start of cutting; the spike at Event 9 marks welding).
2. **Cross-sensor correlation** (not any single sensor) was the key — e.g., "high temperature *and* zero material flow" uniquely identifies welding, distinguishing it from cutting (high T *with* material flow).
3. **Boundary detection** relied on abrupt, multi-sensor transitions rather than gradual drift, which is typical in discrete manufacturing.

The result is a fully labeled process trace that could feed into a discovery algorithm (e.g., the **Alpha algorithm** or **Inductive Miner**) to build a process model from raw, unlabeled sensor data.
