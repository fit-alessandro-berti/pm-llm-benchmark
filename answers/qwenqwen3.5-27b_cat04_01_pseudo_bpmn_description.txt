### **3. Meaning of Gateways**

*   **XOR Gateway (Exclusive Choice):** Represents a decision point where **only one path** can be taken based on a condition.
    *   *Examples:* "Check Request Type," "Is Feasible?", "Is Approval Needed?", "Is Approval Granted?".
    *   *Function:* It acts as a switch. If the condition is true, go one way; if false, go the other.
*   **AND Gateway (Parallel Execution):** Represents a point where the process **splits into multiple paths that must all execute**.
    *   *Example:* "Run Parallel Checks."
    *   *Function:* It ensures that Credit Check and Inventory Check happen concurrently. The process cannot proceed until the **Join** point confirms both are finished.
