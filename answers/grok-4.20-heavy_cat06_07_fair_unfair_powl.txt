```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition
from pm4py.objects.process_tree.obj import Operator

# ============== POWL Model with Potential Unfairness ==============
receive_u = Transition(label="ReceiveApplication")
data_check_u = Transition(label="DataCompletenessCheck")
request_info_u = Transition(label="RequestMoreInfo")
skill_u = Transition(label="SkillAssessment")
cultural_u = Transition(label="CulturalFitCheck")
community_u = Transition(label="CommunityAffiliationCheck")
managerial_u = Transition(label="ManagerialReview")
final_u = Transition(label="FinalDecision")

data_loop_u = OperatorPOWL(operator=Operator.LOOP, children=[data_check_u, request_info_u])
bias_xor = OperatorPOWL(operator=Operator.XOR, children=[cultural_u, community_u])

unfair_nodes = [receive_u, data_loop_u, skill_u, bias_xor, managerial_u, final_u]
unfair_model = StrictPartialOrder(nodes=unfair_nodes)
unfair_model.order.add_edge(receive_u, data_loop_u)
unfair_model.order.add_edge(data_loop_u, skill_u)
unfair_model.order.add_edge(skill_u, bias_xor)
unfair_model.order.add_edge(bias_xor, managerial_u)
unfair_model.order.add_edge(managerial_u, final_u)

# ============== POWL Model Without Unfairness ==============
receive_f = Transition(label="ReceiveApplication")
data_check_f = Transition(label="DataCompletenessCheck")
request_info_f = Transition(label="RequestMoreInfo")
skill_f = Transition(label="SkillAssessment")
cultural_f = Transition(label="CulturalFitCheck")
managerial_f = Transition(label="ManagerialReview")
final_f = Transition(label="FinalDecision")

data_loop_f = OperatorPOWL(operator=Operator.LOOP, children=[data_check_f, request_info_f])

fair_nodes = [receive_f, data_loop_f, skill_f, cultural_f, managerial_f, final_f]
fair_model = StrictPartialOrder(nodes=fair_nodes)
fair_model.order.add_edge(receive_f, data_loop_f)
fair_model.order.add_edge(data_loop_f, skill_f)
fair_model.order.add_edge(skill_f, cultural_f)
fair_model.order.add_edge(cultural_f, managerial_f)
fair_model.order.add_edge(managerial_f, final_f)
```

**Explanation** (for clarity, not part of the models):
- **Unfair model**: After `SkillAssessment` there is an `XOR` that can route to `CommunityAffiliationCheck` (potential bias point). The loop handles data completeness.
- **Fair model**: All applicants go through the identical `CulturalFitCheck`; the biased `XOR` branch and `CommunityAffiliationCheck` activity are removed.  
Both models follow the described sequence, loop for data completeness, and managerial/final steps.
