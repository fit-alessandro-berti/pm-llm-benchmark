```python
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Activities
OR = Transition(label="OR")      # Order Received
MS = Transition(label="MS")      # Material Sourcing
QS = Transition(label="QS")      # Quality Screening
AS = Transition(label="AS")      # Assembly
IQC = Transition(label="IQC")    # In-line Quality Check
TST = Transition(label="TST")    # Testing
RT = Transition(label="RT")      # Re-Testing
PK = Transition(label="PK")      # Packaging
LB = Transition(label="LB")      # Labeling
DP = Transition(label="DP")      # Distribution Planning
D = Transition(label="D")        # Dispatch

tau = SilentTransition()

# Assembly may be followed by repeated IQC + Assembly cycles.
# This captures:
# - AS only
# - AS, IQC, AS
# - AS, IQC, AS, IQC, AS
# etc.
assembly_iqc_loop = OperatorPOWL(
    operator=Operator.LOOP,
    children=[
        AS,
        IQC
    ]
)

# Testing may be followed by repeated RT + TST cycles.
# This captures:
# - TST only
# - TST, RT, TST
# - TST, RT, TST, RT, TST
# etc.
testing_retest_loop = OperatorPOWL(
    operator=Operator.LOOP,
    children=[
        TST,
        RT
    ]
)

# Labeling is optional:
# either LB happens, or it is skipped silently.
optional_labeling = OperatorPOWL(
    operator=Operator.XOR,
    children=[
        LB,
        tau
    ]
)

# Overall process partial order
root = StrictPartialOrder(
    nodes=[
        OR,
        MS,
        QS,
        assembly_iqc_loop,
        testing_retest_loop,
        PK,
        optional_labeling,
        DP,
        D
    ]
)

# Main sequential dependencies
root.order.add_edge(OR, MS)
root.order.add_edge(MS, QS)
root.order.add_edge(QS, assembly_iqc_loop)
root.order.add_edge(assembly_iqc_loop, testing_retest_loop)
root.order.add_edge(testing_retest_loop, PK)
root.order.add_edge(PK, optional_labeling)
root.order.add_edge(optional_labeling, DP)
root.order.add_edge(DP, D)
```

Equivalent compact POWL notation:

```text
PO=(
  nodes={
    OR,
    MS,
    QS,
    *(AS, IQC),
    *(TST, RT),
    PK,
    X(LB, tau),
    DP,
    D
  },
  order={
    OR --> MS,
    MS --> QS,
    QS --> *(AS, IQC),
    *(AS, IQC) --> *(TST, RT),
    *(TST, RT) --> PK,
    PK --> X(LB, tau),
    X(LB, tau) --> DP,
    DP --> D
  }
)
```

This model captures the observed behavior:

- Every case starts with `OR`, followed by `MS`, then `QS`.
- Assembly is mandatory, while `IQC` may occur zero or more times between assembly repetitions.
- Testing is mandatory, while `RT` may occur zero or more times, each time leading back to another `TST`.
- Packaging occurs after testing is completed.
- Labeling is optional.
- Distribution planning and dispatch happen at the end.
