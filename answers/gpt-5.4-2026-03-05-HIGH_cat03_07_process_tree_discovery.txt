A concise final process tree is:

```text
->(
  'Receive Application (RA)',
  'Validate Form (VF)',
  'Assess Eligibility (AE)',
  *(
    tau,
    ->(
      'Request Missing Info (RMI)',
      'Validate Form (VF)'
    )
  ),
  'Approval Decision (AD)',
  'Send Confirmation (SC)',
  'Archive Case (AC)'
)
```

This models the main sequence, with an optional loop after eligibility assessment that can repeat the cycle `Request Missing Info -> Validate Form` zero or more times before continuing to approval, confirmation, and archiving.
