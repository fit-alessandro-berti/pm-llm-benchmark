```sql
WITH valid_cases AS (
    SELECT
        case_id
    FROM event_log
    GROUP BY case_id
    HAVING COUNT(*) <= 6
)
SELECT
    e.*
FROM event_log e
JOIN valid_cases v
    ON e.case_id = v.case_id
ORDER BY e.case_id, e.timestamp;
```

This works as follows:

- `GROUP BY case_id` collects all events belonging to the same case.
- `HAVING COUNT(*) <= 6` keeps only cases with six or fewer events.
- The final `JOIN` returns all original event rows for those valid cases.
