Below are two POWL models reflecting the GlobalTeq Solutions hiring process.  
Both models include:

- **ReceiveApplication**  
- **DataCompletenessCheck**  
- **RequestMoreInfo** (used in the loop for incomplete data)  
- **SkillAssessment**  
- **CulturalFitCheck**  
- **ManagerialReview**  
- **FinalDecision**

The first model adds **CommunityAffiliationCheck** in an XOR branch after **SkillAssessment**, creating a potential source of unfairness.  
The second model removes this XOR branch and uses a single, uniform **CulturalFitCheck** for all applicants.

---

### 1. POWL Model with Potential Unfairness

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Basic activities
ReceiveApplication = Transition(label="ReceiveApplication")
DataCompletenessCheck = Transition(label="DataCompletenessCheck")
RequestMoreInfo = Transition(label="RequestMoreInfo")
SkillAssessment = Transition(label="SkillAssessment")
CulturalFitCheck = Transition(label="CulturalFitCheck")
CommunityAffiliationCheck = Transition(label="CommunityAffiliationCheck")
ManagerialReview = Transition(label="ManagerialReview")
FinalDecision = Transition(label="FinalDecision")
skip = SilentTransition()

# Loop for data completeness: check -> if incomplete -> request more info -> check again
loop_data = OperatorPOWL(
    operator=Operator.LOOP,
    children=[DataCompletenessCheck, RequestMoreInfo]
)

# XOR after SkillAssessment: either standard CulturalFitCheck OR CommunityAffiliationCheck
xor_cultural = OperatorPOWL(
    operator=Operator.XOR,
    children=[CulturalFitCheck, CommunityAffiliationCheck]
)

# Partial order for the main process
nodes = [
    ReceiveApplication,
    loop_data,
    SkillAssessment,
    xor_cultural,
    ManagerialReview,
    FinalDecision
]
root_unfair = StrictPartialOrder(nodes=nodes)
root_unfair.order.add_edge(ReceiveApplication, loop_data)
root_unfair.order.add_edge(loop_data, SkillAssessment)
root_unfair.order.add_edge(SkillAssessment, xor_cultural)
root_unfair.order.add_edge(xor_cultural, ManagerialReview)
root_unfair.order.add_edge(ManagerialReview, FinalDecision)
```

- **Where unfairness can appear**:  
  The XOR branch after **SkillAssessment** allows some applicants to take **CommunityAffiliationCheck**, which can provide a subtle advantage. This path is not available to all, potentially biasing later stages (e.g., **ManagerialReview**) in favor of affiliated or local candidates.

---

### 2. POWL Model Without Unfairness

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Basic activities (same as before, but without CommunityAffiliationCheck)
ReceiveApplication = Transition(label="ReceiveApplication")
DataCompletenessCheck = Transition(label="DataCompletenessCheck")
RequestMoreInfo = Transition(label="RequestMoreInfo")
SkillAssessment = Transition(label="SkillAssessment")
CulturalFitCheck = Transition(label="CulturalFitCheck")
ManagerialReview = Transition(label="ManagerialReview")
FinalDecision = Transition(label="FinalDecision")
skip = SilentTransition()

# Loop for data completeness (unchanged)
loop_data = OperatorPOWL(
    operator=Operator.LOOP,
    children=[DataCompletenessCheck, RequestMoreInfo]
)

# Partial order: all applicants follow the same path
nodes_fair = [
    ReceiveApplication,
    loop_data,
    SkillAssessment,
    CulturalFitCheck,
    ManagerialReview,
    FinalDecision
]
root_fair = StrictPartialOrder(nodes=nodes_fair)
root_fair.order.add_edge(ReceiveApplication, loop_data)
root_fair.order.add_edge(loop_data, SkillAssessment)
root_fair.order.add_edge(SkillAssessment, CulturalFitCheck)
root_fair.order.add_edge(CulturalFitCheck, ManagerialReview)
root_fair.order.add_edge(ManagerialReview, FinalDecision)
```

- **Fairness property**:  
  There is no XOR branch providing selective advantages based on community affiliation. All candidates undergo the same **CulturalFitCheck**, ensuring consistent treatment at that stage.
