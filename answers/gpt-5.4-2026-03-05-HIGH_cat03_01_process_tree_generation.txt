A suitable process tree is:

\[
->(
'A',
'RG',
'IA',
'TI',
'TC',
X('CE','FBP'),
*(\tau,\ ->('SC', X('TM','SPT'), 'RDT', X('FCE','FBA'))),
+('AI','NC'),
'FAS',
'DDI',
+('BL','ICP','MRF')
)
\]

### Reading of the tree
- `A -> RG -> IA -> TI -> TC`: arrival, registration, intake, triage interview, triage classification
- `X('CE','FBP')`: exactly one initial diagnostic test
- `*(tau, ->(...))`: zero or more abnormal-result treatment cycles  
  - `tau` allows immediate exit if results are normal after the initial test
  - otherwise repeat:
    - `SC`
    - `X('TM','SPT')`
    - `RDT`
    - `X('FCE','FBA')`
- `+('AI','NC')`: advanced imaging and nutritional counseling in parallel
- `FAS -> DDI`: final administrative/information sequence
- `+('BL','ICP','MRF')`: final clearance activities in parallel

---

## PM4Py construction

```python
from pm4py.objects.process_tree.obj import ProcessTree, Operator

# Root sequence
root = ProcessTree(operator=Operator.SEQUENCE)

# Simple sequential leaves
A = ProcessTree(label="A", parent=root)
RG = ProcessTree(label="RG", parent=root)
IA = ProcessTree(label="IA", parent=root)
TI = ProcessTree(label="TI", parent=root)
TC = ProcessTree(label="TC", parent=root)

# Initial diagnostic XOR: CE or FBP
init_diag = ProcessTree(operator=Operator.XOR, parent=root)
CE = ProcessTree(label="CE", parent=init_diag)
FBP = ProcessTree(label="FBP", parent=init_diag)
init_diag.children.extend([CE, FBP])

# Loop for zero or more abnormal-result cycles
# *(tau, ->(SC, X(TM,SPT), RDT, X(FCE,FBA)))
loop = ProcessTree(operator=Operator.LOOP, parent=root)

tau = ProcessTree(parent=loop)  # silent step

redo_seq = ProcessTree(operator=Operator.SEQUENCE, parent=loop)
SC = ProcessTree(label="SC", parent=redo_seq)

treat_xor = ProcessTree(operator=Operator.XOR, parent=redo_seq)
TM = ProcessTree(label="TM", parent=treat_xor)
SPT = ProcessTree(label="SPT", parent=treat_xor)
treat_xor.children.extend([TM, SPT])

RDT = ProcessTree(label="RDT", parent=redo_seq)

follow_xor = ProcessTree(operator=Operator.XOR, parent=redo_seq)
FCE = ProcessTree(label="FCE", parent=follow_xor)
FBA = ProcessTree(label="FBA", parent=follow_xor)
follow_xor.children.extend([FCE, FBA])

redo_seq.children.extend([SC, treat_xor, RDT, follow_xor])
loop.children.extend([tau, redo_seq])

# Parallel block: AI and NC
post_diag_parallel = ProcessTree(operator=Operator.PARALLEL, parent=root)
AI = ProcessTree(label="AI", parent=post_diag_parallel)
NC = ProcessTree(label="NC", parent=post_diag_parallel)
post_diag_parallel.children.extend([AI, NC])

# Final sequential steps
FAS = ProcessTree(label="FAS", parent=root)
DDI = ProcessTree(label="DDI", parent=root)

# Final parallel administrative block
final_parallel = ProcessTree(operator=Operator.PARALLEL, parent=root)
BL = ProcessTree(label="BL", parent=final_parallel)
ICP = ProcessTree(label="ICP", parent=final_parallel)
MRF = ProcessTree(label="MRF", parent=final_parallel)
final_parallel.children.extend([BL, ICP, MRF])

# Assemble root
root.children.extend([
    A,
    RG,
    IA,
    TI,
    TC,
    init_diag,
    loop,
    post_diag_parallel,
    FAS,
    DDI,
    final_parallel
])
```
