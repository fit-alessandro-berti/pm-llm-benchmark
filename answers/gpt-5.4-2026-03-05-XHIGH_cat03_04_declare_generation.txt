```python
# Activity codes:

# CE  = Cost Evaluation

# LT  = Laboratory Testing

# AG  = Approval Gate
# MP  = Marketing Plan
# FL  = Final Launch

rule = {"support": 1.0, "confidence": 1.0}

declare_model = {
    "existence": {

        "TFC": rule.copy(),
        "CE": rule.copy(),
        "PC": rule.copy(),
        "LT": rule.copy(),
        "UT": rule.copy(),

    },

        ("LT", "UT"): rule.copy(),
    },
    "response": {},
    "precedence": {},
    "succession": {

```

This assumes a successful product path where:
- `IG` starts the case,
- `DD` is followed by both `TFC` and `CE`,
- `PC` happens after both checks,
- both `LT` and `UT` are required before `AG`,
- then `MP`,
- then `FL`.
