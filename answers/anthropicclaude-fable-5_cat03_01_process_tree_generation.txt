# Process Tree Model

## Tree Notation

```
-> ( 'A', 
     'RG', 
     'IA', 
     'TI', 
     'TC', 
     X ( 'CE', 'FBP' ), 
     * ( tau, 
         -> ( 'SC', 
              X ( 'TM', 'SPT' ), 
              X ( 'FCE', 'FBA' ) 
            ) 
       ), 
     + ( 'AI', 'NC' ), 
     'FAS', 
     'DDI', 
     + ( 'BL', 'ICP', 'MRF' ) 
   )
```

**Explanation of the loop:** The `*` operator first executes its "do" part (here `tau`, a silent step). At this point we may exit (results normal) or execute the "redo" part — the sequence of Specialist Consultation, exclusive treatment choice (TM or SPT), and re-diagnostic test (FCE or FBA) — and then return to the decision point. This repeats until results are normal.

## Python Construction (pm4py)

```python
from pm4py.objects.process_tree.obj import ProcessTree, Operator

# Root: overall sequence
root = ProcessTree(operator=Operator.SEQUENCE)

# 1-2. Arrival, Registration, Intake, Triage Interview, Triage Classification
A  = ProcessTree(label="A",  parent=root)
RG = ProcessTree(label="RG", parent=root)
IA = ProcessTree(label="IA", parent=root)
TI = ProcessTree(label="TI", parent=root)
TC = ProcessTree(label="TC", parent=root)

# 3. Exclusive choice of initial diagnostic test
xor_init = ProcessTree(operator=Operator.XOR, parent=root)
CE  = ProcessTree(label="CE",  parent=xor_init)
FBP = ProcessTree(label="FBP", parent=xor_init)
xor_init.children.append(CE)
xor_init.children.append(FBP)

# 4. Treatment loop: * ( tau, -> ( SC, X(TM, SPT), X(FCE, FBA) ) )
loop = ProcessTree(operator=Operator.LOOP, parent=root)

tau = ProcessTree(parent=loop)  # silent step (no label, no operator)

redo_seq = ProcessTree(operator=Operator.SEQUENCE, parent=loop)
SC = ProcessTree(label="SC", parent=redo_seq)

xor_treat = ProcessTree(operator=Operator.XOR, parent=redo_seq)
TM  = ProcessTree(label="TM",  parent=xor_treat)
SPT = ProcessTree(label="SPT", parent=xor_treat)
xor_treat.children.append(TM)
xor_treat.children.append(SPT)

xor_rdt = ProcessTree(operator=Operator.XOR, parent=redo_seq)
FCE = ProcessTree(label="FCE", parent=xor_rdt)
FBA = ProcessTree(label="FBA", parent=xor_rdt)
xor_rdt.children.append(FCE)
xor_rdt.children.append(FBA)

redo_seq.children.append(SC)
redo_seq.children.append(xor_treat)
redo_seq.children.append(xor_rdt)

loop.children.append(tau)
loop.children.append(redo_seq)

# 5. Parallel: Advanced Imaging + Nutritional Counseling
par_support = ProcessTree(operator=Operator.PARALLEL, parent=root)
AI = ProcessTree(label="AI", parent=par_support)
NC = ProcessTree(label="NC", parent=par_support)
par_support.children.append(AI)
par_support.children.append(NC)

# 6. Final informational steps
FAS = ProcessTree(label="FAS", parent=root)
DDI = ProcessTree(label="DDI", parent=root)

# 7. Parallel administrative/financial clearance
par_admin = ProcessTree(operator=Operator.PARALLEL, parent=root)
BL  = ProcessTree(label="BL",  parent=par_admin)
ICP = ProcessTree(label="ICP", parent=par_admin)
MRF = ProcessTree(label="MRF", parent=par_admin)
par_admin.children.append(BL)
par_admin.children.append(ICP)
par_admin.children.append(MRF)

# Assemble root children in order
root.children.append(A)
root.children.append(RG)
root.children.append(IA)
root.children.append(TI)
root.children.append(TC)
root.children.append(xor_init)
root.children.append(loop)
root.children.append(par_support)
root.children.append(FAS)
root.children.append(DDI)
root.children.append(par_admin)
```

## How the model matches the description

| Step | Construct |
|------|-----------|
| 1–2. Arrival  Registration  Intake  Triage Interview  Classification | Sequence of leaves `A, RG, IA, TI, TC` |
| 3. Initial test (one of two) | `X('CE', 'FBP')` |
| 4. Repeating consultation/treatment/re-test until normal | `*(tau, ->('SC', X('TM','SPT'), X('FCE','FBA')))` — zero or more iterations, exiting when results are normal |
| 5. Imaging and counseling in any order/parallel | `+('AI', 'NC')` |
| 6. Scheduling then discharge instructions | Sequence `FAS, DDI` |
| 7. Three concurrent administrative clearances | `+('BL', 'ICP', 'MRF')` |
