A suitable process tree is:

```text
->(
  'A',
  'RG',
  'IA',
  'TI',
  'TC',
  X('CE', 'FBP'),
  X(
    tau,
    *(
      ->(
        'SC',
        X('TM', 'SPT'),
        'RDT',
        X('FCE', 'FBA')
      ),
      tau
    )
  ),
  +('AI', 'NC'),
  'FAS',
  'DDI',
  +('BL', 'ICP', 'MRF')
)
```

Here, the `X(tau, *(...))` part models that the abnormal-results loop may be skipped if the initial diagnostic result is already normal. If abnormal, the loop executes one or more rounds of:

```text
SC -> X(TM, SPT) -> RDT -> X(FCE, FBA)
```

until normal results are obtained.

A corresponding PM4Py construction is:

```python
from pm4py.objects.process_tree.obj import ProcessTree, Operator

def act(label):
    return ProcessTree(label=label)

def tau():
    return ProcessTree()  # silent step

def node(operator, *children):
    n = ProcessTree(operator=operator)
    for c in children:
        c.parent = n
        n.children.append(c)
    return n

rdt_step = node(
    Operator.SEQUENCE,
    act("RDT"),
    node(Operator.XOR, act("FCE"), act("FBA"))
)

abnormal_cycle = node(
    Operator.SEQUENCE,
    act("SC"),
    node(Operator.XOR, act("TM"), act("SPT")),
    rdt_step
)

abnormal_loop = node(
    Operator.LOOP,
    abnormal_cycle,
    tau()
)

optional_abnormal_loop = node(
    Operator.XOR,
    tau(),
    abnormal_loop
)

root = node(
    Operator.SEQUENCE,
    act("A"),
    act("RG"),
    act("IA"),
    act("TI"),
    act("TC"),
    node(Operator.XOR, act("CE"), act("FBP")),
    optional_abnormal_loop,
    node(Operator.PARALLEL, act("AI"), act("NC")),
    act("FAS"),
    act("DDI"),
    node(Operator.PARALLEL, act("BL"), act("ICP"), act("MRF"))
)
```
