Here are the two POWL models for the hiring process at **GlobalTeq Solutions**, reflecting the described scenarios with and without unfairness:

---

### **1. POWL Model with Potential Unfairness**
This model includes an **XOR branch** where applicants with local community affiliations receive a subtle advantage.

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Activities
receive_application = Transition(label="ReceiveApplication")
data_completeness_check = Transition(label="DataCompletenessCheck")
request_more_info = Transition(label="RequestMoreInfo")
skill_assessment = Transition(label="SkillAssessment")
standard_cultural_fit = Transition(label="StandardCulturalFitCheck")
community_affiliation_check = Transition(label="CommunityAffiliationCheck")
managerial_review = Transition(label="ManagerialReview")
final_decision = Transition(label="FinalDecision")

# Loop for data completeness
data_loop = OperatorPOWL(
    operator=Operator.LOOP,
    children=[data_completeness_check, request_more_info]
)

# XOR branch: Standard vs. Community Affiliation Check
cultural_fit_xor = OperatorPOWL(
    operator=Operator.XOR,
    children=[
        standard_cultural_fit,  # Standard path
        community_affiliation_check  # Community-affiliated path (with bias)
    ]
)

# Root workflow with unfairness
root = StrictPartialOrder(nodes=[
    receive_application,
    data_loop,
    skill_assessment,
    cultural_fit_xor,
    managerial_review,
    final_decision
])

# Define partial order (sequential dependencies)
root.order.add_edge(receive_application, data_loop)
root.order.add_edge(data_loop, skill_assessment)
root.order.add_edge(skill_assessment, cultural_fit_xor)
root.order.add_edge(cultural_fit_xor, managerial_review)
root.order.add_edge(managerial_review, final_decision)
```

**Key Bias Point:**
- The **XOR branch** (`cultural_fit_xor`) allows applicants to bypass standard cultural fit evaluation if they are affiliated with a local group, introducing unfairness.

---

### **2. POWL Model Without Unfairness**
This model ensures **all applicants undergo the same cultural fit evaluation**, eliminating bias.

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Activities
receive_application = Transition(label="ReceiveApplication")
data_completeness_check = Transition(label="DataCompletenessCheck")
request_more_info = Transition(label="RequestMoreInfo")
skill_assessment = Transition(label="SkillAssessment")
cultural_fit_check = Transition(label="CulturalFitCheck")  # Unified evaluation
managerial_review = Transition(label="ManagerialReview")
final_decision = Transition(label="FinalDecision")

# Loop for data completeness
data_loop = OperatorPOWL(
    operator=Operator.LOOP,
    children=[data_completeness_check, request_more_info]
)

# Root workflow (fair)
root = StrictPartialOrder(nodes=[
    receive_application,
    data_loop,
    skill_assessment,
    cultural_fit_check,  # Single path for all applicants
    managerial_review,
    final_decision
])

# Define partial order
root.order.add_edge(receive_application, data_loop)
root.order.add_edge(data_loop, skill_assessment)
root.order.add_edge(skill_assessment, cultural_fit_check)
root.order.add_edge(cultural_fit_check, managerial_review)
root.order.add_edge(managerial_review, final_decision)
```

**Key Fairness Guarantee:**
- **No XOR branch** exists; all applicants undergo the same **cultural fit evaluation**, ensuring fairness.

---
### **Key Differences:**
| Model | Bias Risk? | Cultural Fit Path | Loop Handling |
|--------|------------|-------------------|---------------|
| **With Unfairness** |  Yes | XOR: Standard or Community Affiliation Check | Data completeness loop |
| **Without Unfairness** |  No | Single unified path | Data completeness loop |

This ensures the second model adheres to **meritocratic hiring principles** while the first demonstrates how bias could creep in via **implicit affiliation-based advantages**.
