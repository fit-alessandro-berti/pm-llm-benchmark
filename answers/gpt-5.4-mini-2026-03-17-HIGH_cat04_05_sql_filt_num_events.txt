```sql
WITH eligible_cases AS (
    SELECT case_id
    FROM event_log
    GROUP BY case_id
    HAVING COUNT(*) <= 6
)
SELECT e.*
FROM event_log AS e
JOIN eligible_cases AS c
  ON e.case_id = c.case_id
ORDER BY e.case_id, e.timestamp;
```

### How it works
- `GROUP BY case_id` counts events per case.
- `HAVING COUNT(*) <= 6` keeps only cases with six or fewer events.
- The final `JOIN` returns all events belonging to those eligible cases.
