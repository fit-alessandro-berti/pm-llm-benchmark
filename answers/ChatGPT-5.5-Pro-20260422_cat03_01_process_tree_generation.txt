A compact readable process tree for the whole hospital process is:

```text
->(
  'A',
  'RG',
  'IA',
  'TI',
  'TC',
  X(
    'CE',
    'FBP'
  ),
  X(
    tau,
    *(
      ->(
        'SC',
        X(
          'TM',
          'SPT'
        ),
        'RDT',
        X(
          'FCE',
          'FBA'
        )
      ),
      tau
    )
  ),
  +(
    'AI',
    'NC'
  ),
  'FAS',
  'DDI',
  +(
    'BL',
    'ICP',
    'MRF'
  )
)
```

Here, the part

```text
X(
  tau,
  *(
    ->('SC', X('TM', 'SPT'), 'RDT', X('FCE', 'FBA')),
    tau
  )
)
```

models the abnormal-results cycle. The `tau` branch means the first diagnostic result was already normal, so the loop is skipped. Otherwise, the patient goes through specialist consultation, one treatment choice, re-diagnostic testing, and repeats as needed.

If you need strict binary notation, the same tree can be written as:

```text
->(
  'A',
  ->(
    'RG',
    ->(
      'IA',
      ->(
        'TI',
        ->(
          'TC',
          ->(
            X('CE', 'FBP'),
            ->(
              X(
                tau,
                *(
                  ->(
                    'SC',
                    ->(
                      X('TM', 'SPT'),
                      ->(
                        'RDT',
                        X('FCE', 'FBA')
                      )
                    )
                  ),
                  tau
                )
              ),
              ->(
                +('AI', 'NC'),
                ->(
                  'FAS',
                  ->(
                    'DDI',
                    +(
                      'BL',
                      +('ICP', 'MRF')
                    )
                  )
                )
              )
            )
          )
        )
      )
    )
  )
)
```

A corresponding PM4Py construction is:

```python
from pm4py.objects.process_tree.obj import ProcessTree, Operator


def leaf(label):
    return ProcessTree(label=label)


def tau():
    # In PM4Py, a silent leaf is a ProcessTree node without operator and without label.
    return ProcessTree()


def node(operator, children):
    parent = ProcessTree(operator=operator)
    for child in children:
        child.parent = parent
        parent.children.append(child)
    return parent


# Initial diagnostic choice
initial_diagnostics = node(Operator.XOR, [
    leaf("CE"),
    leaf("FBP")
])

# One abnormal-results treatment/re-diagnostic cycle
abnormal_cycle = node(Operator.SEQUENCE, [
    leaf("SC"),
    node(Operator.XOR, [
        leaf("TM"),
        leaf("SPT")
    ]),
    leaf("RDT"),
    node(Operator.XOR, [
        leaf("FCE"),
        leaf("FBA")
    ])
])

# Optional abnormal loop:
# tau = skip loop when results are normal immediately
# LOOP(abnormal_cycle, tau) = repeat abnormal cycle one or more times
abnormal_handling = node(Operator.XOR, [
    tau(),
    node(Operator.LOOP, [
        abnormal_cycle,
        tau()
    ])
])

# Advanced support in true concurrency
advanced_support = node(Operator.PARALLEL, [
    leaf("AI"),
    leaf("NC")
])

# Final administrative/financial clearances in true concurrency
final_clearance = node(Operator.PARALLEL, [
    leaf("BL"),
    leaf("ICP"),
    leaf("MRF")
])

# Full process tree
root = node(Operator.SEQUENCE, [
    leaf("A"),
    leaf("RG"),
    leaf("IA"),
    leaf("TI"),
    leaf("TC"),
    initial_diagnostics,
    abnormal_handling,
    advanced_support,
    leaf("FAS"),
    leaf("DDI"),
    final_clearance
])
```
