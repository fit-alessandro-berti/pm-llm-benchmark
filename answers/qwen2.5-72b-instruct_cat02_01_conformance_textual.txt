### Case Analysis and Violations

Let's analyze each case and identify any deviations from the normative process described.

#### Case 1
**Activities:**
1. Receive order
2. Check inventory
3. Authorize payment
4. Package items
5. Ship order
6. Close order

**Analysis:**
- This case follows the expected sequence for an order with sufficient inventory.
- There are no violations.

#### Case 2
**Activities:**
1. Receive order
2. Check inventory
3. Package items
4. Authorize payment
5. Ship order
6. Close order

**Analysis:**
- **Violation:** Payment authorization should occur before packaging items.
- **Explanation:** According to the normative process, "Package items" should only occur after "Authorize payment" because items must be prepared after ensuring the payment is valid.

#### Case 3
**Activities:**
1. Receive order
2. Check inventory
3. Authorize payment
4. Cancel order
5. Ship order
6. Close order

**Analysis:**
- **Violation:** Shipping an order after canceling it.
- **Explanation:** Once an order is canceled, it is illogical to proceed with activities intended for fulfillment. "Ship order" should not occur after "Cancel order."

#### Case 4
**Activities:**
1. Receive order
2. Check inventory
3. Cancel order
4. Close order

**Analysis:**
- This case follows the expected sequence for an order with insufficient inventory.
- There are no violations.

#### Case 5
**Activities:**
1. Receive order
2. Check inventory
3. Authorize payment
4. Ship order
5. Close order

**Analysis:**
- **Violation:** Items were shipped without being packaged.
- **Explanation:** According to the normative process, "Package items" should occur before "Ship order" because items must be prepared before they can be shipped.

### Summary of Violations

1. **Case 2:**
   - **Violation:** Payment authorization occurred after packaging items.
   - **Explanation:** "Authorize payment" should occur before "Package items."

2. **Case 3:**
   - **Violation:** Shipping an order after canceling it.
   - **Explanation:** "Ship order" should not occur after "Cancel order."

3. **Case 5:**
   - **Violation:** Items were shipped without being packaged.
   - **Explanation:** "Package items" should occur before "Ship order."

### Conclusion

Cases 1 and 4 follow the normative process without any violations. Cases 2, 3, and 5 contain deviations from the expected sequence of activities, as detailed above.
