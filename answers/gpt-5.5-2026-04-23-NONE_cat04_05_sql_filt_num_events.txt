```sql
SELECT
    e.*
FROM event_log AS e
JOIN (
    SELECT
        case_id
    FROM event_log
    GROUP BY case_id
    HAVING COUNT(*) <= 6
) AS valid_cases
    ON e.case_id = valid_cases.case_id;
```

This query:

1. Groups events by `case_id`.
2. Counts the number of events in each case.
3. Keeps only cases with `COUNT(*) <= 6`.
4. Joins those valid `case_id` values back to `event_log` to return all original event rows for those cases.
