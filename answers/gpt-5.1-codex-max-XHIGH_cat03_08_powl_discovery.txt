Below is a POWL that fits all observed variants (mandatory linear skeleton, assembly with zero-or-more inline checks, test/retest loop, optional labeling):

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Atomic transitions
t_OR  = Transition("OR")
t_MS  = Transition("MS")
t_QS  = Transition("QS")
t_AS  = Transition("AS")
t_IQC = Transition("IQC")
t_TST = Transition("TST")
t_RT  = Transition("RT")
t_PK  = Transition("PK")
t_LB  = Transition("LB")
t_DP  = Transition("DP")
t_D   = Transition("D")

# Silent steps
tau_iqc  = SilentTransition()  # allows zero-or-more IQC after an AS
tau_skip = SilentTransition()  # allows skipping LB

# Inner loop: zero-or-more IQC
inner_iqc_loop = OperatorPOWL(operator=Operator.LOOP, children=[tau_iqc, t_IQC])
# Outer loop: one-or-more AS, each followed by zero-or-more IQC
assembly_loop  = OperatorPOWL(operator=Operator.LOOP, children=[t_AS, inner_iqc_loop])

# Test / re-test loop
test_loop = OperatorPOWL(operator=Operator.LOOP, children=[t_TST, t_RT])

# Optional labeling
lb_choice = OperatorPOWL(operator=Operator.XOR, children=[t_LB, tau_skip])

# Global partial order (sequential chain here)
root = StrictPartialOrder(nodes=[
    t_OR, t_MS, t_QS,
    assembly_loop,
    test_loop,
    t_PK,
    lb_choice,
    t_DP,
    t_D
])

root.order.add_edge(t_OR, t_MS)
root.order.add_edge(t_MS, t_QS)
root.order.add_edge(t_QS, assembly_loop)
root.order.add_edge(assembly_loop, test_loop)
root.order.add_edge(test_loop, t_PK)
root.order.add_edge(t_PK, lb_choice)
root.order.add_edge(lb_choice, t_DP)
root.order.add_edge(t_DP, t_D)
```

- `assembly_loop = * ( AS , * (  , IQC ) )` covers: AS alone, ASIQC, ASIQCIQC, and multiple AS/IQC cycles.
- `test_loop = * ( TST , RT )` covers repeated re-tests.
- `lb_choice = X ( LB ,  )` makes labeling optional.
- The StrictPartialOrder edges enforce the observed linear order: OR  MS  QS  (assembly loop)  (test loop)  PK  (LB optional)  DP  D.
