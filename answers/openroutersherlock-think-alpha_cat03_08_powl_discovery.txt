```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Basic activities
OR = Transition(label="OR")
MS = Transition(label="MS")
QS = Transition(label="QS")
AS = Transition(label="AS")
IQC = Transition(label="IQC")
TST = Transition(label="TST")
RT = Transition(label="RT")
PK = Transition(label="PK")
LB = Transition(label="LB")
DP = Transition(label="DP")
D = Transition(label="D")
tau = SilentTransition()

# IQC loop: one or more IQC (handles consecutive IQCs like IQC IQC)
iqc_loop = OperatorPOWL(operator=Operator.LOOP, children=[IQC, tau])

# Assembly loop: AS (iqc_loop AS)* (always starts and ends with AS, optional IQC blocks)
assembly_loop = OperatorPOWL(operator=Operator.LOOP, children=[AS, iqc_loop])

# Testing loop: TST (RT TST)* 
testing_loop = OperatorPOWL(operator=Operator.LOOP, children=[TST, RT])

# Optional LB: choice between LB or skip (tau)
opt_lb = OperatorPOWL(operator=Operator.XOR, children=[LB, tau])

# Root partial order enforcing the sequence OR -> MS -> QS -> assembly -> testing -> PK -> opt_lb -> DP -> D
root = StrictPartialOrder(nodes=[OR, MS, QS, assembly_loop, testing_loop, PK, opt_lb, DP, D])
root.order.add_edge(OR, MS)
root.order.add_edge(MS, QS)
root.order.add_edge(QS, assembly_loop)
root.order.add_edge(assembly_loop, testing_loop)
root.order.add_edge(testing_loop, PK)
root.order.add_edge(PK, opt_lb)
root.order.add_edge(opt_lb, DP)
root.order.add_edge(DP, D)
```

This POWL model captures the process:
- Fixed prefix: OR  MS  QS
- Assembly phase `assembly_loop`: Always at least one AS; optional blocks of one-or-more IQC followed by another AS (e.g., AS, AS IQC AS, AS IQC IQC AS IQC AS)
- Testing phase `testing_loop`: At least one TST; optional RT  TST repeats (e.g., TST, TST RT TST)
- Packaging: PK  optional LB (via XOR with tau)  DP  D
