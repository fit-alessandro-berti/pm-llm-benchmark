```python
# Example of how to check for a deviation given ZETA = 2
def check_deviation(activity_pair, observed_time, profile, zeta=2):
    if activity_pair not in profile:
        return "Pair not in profile"
    
    avg, stdev = profile[activity_pair]
    lower_bound = avg - (zeta * stdev)
    upper_bound = avg + (zeta * stdev)
    
    if observed_time < lower_bound or observed_time > upper_bound:
        return True  # Deviation occurred
    return False # Within normal bounds
```

### Rationale for the estimated values:
1.  **The "Lead Time Gap"**: The largest jump is between `OP` and `RC`, reflecting the time components take to be manufactured and shipped from a global supplier.
2.  **The "Quality/Assembly" Phase**: These are modeled as internal factory processes with tighter standard deviations, as they are more controllable than international shipping.
3.  **The "Support" Lag**: The time between `DT` (Distribution) and `AS` (After-Sales) is the longest, as it represents the time the customer actually uses the product before needing support.
4.  **Cumulative Variance**: For distant pairs (e.g., `SS` to `AS`), the standard deviation is significantly higher, reflecting the aggregation of uncertainties across multiple process steps.
