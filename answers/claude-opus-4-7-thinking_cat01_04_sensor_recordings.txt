# Process Mining Analysis: Manufacturing Assembly Line

## Pattern Analysis & Activity Labeling

After examining the sensor readings, I identified **six distinct phases** based on transitions in temperature, pressure, vibration, material flow, tool position, and energy consumption.

---

###  **Phase 1: Idle / Startup (Events 1–3)**
| Signature | Values |
|---|---|
| Temperature | 20–21 °C (ambient) |
| Pressure | 1 bar |
| Vibration | 0 Hz |
| Flow / Tool Pos. | 0 / 0 |
| Energy | 0.40–0.50 kWh |

**Rationale:** All readings are at baseline — ambient temperature, no vibration, no material flow, and minimal energy draw. This indicates the machine is **idle or warming up** before production begins.

---

###  **Phase 2: Cutting Metal (Events 4–6)**
| Signature | Values |
|---|---|
| Temperature | 50  58 °C (rising) |
| Pressure | 1  1.5 bar |
| Vibration | 30–38 Hz (high) |
| Flow | 5 units/s (steady feed) |
| Tool Position | 5  10 mm (advancing) |
| Energy | 2.0  3.0 kWh (climbing) |

**Rationale:** A sharp jump in temperature, high vibration, steady material flow, and progressively advancing tool position are classic signs of a **cutting operation**. The tool is physically engaging the material, generating friction heat and mechanical oscillation.

---

###  **Phase 3: Assembling Parts (Events 7–8)**
| Signature | Values |
|---|---|
| Temperature | 40 °C (moderate, stable) |
| Pressure | 2 bar (firm, steady) |
| Vibration | 10 Hz (low) |
| Flow | 2 units/s |
| Tool Position | 15 mm (fixed) |
| Energy | 2.0 kWh (stable) |

**Rationale:** Vibration has dropped substantially while pressure has increased slightly — typical of a **pressing/assembly step** where parts are joined together. The tool position is stationary and readings are steady, suggesting a controlled mating operation rather than cutting.

---

###  **Phase 4: Welding (Event 9)**
| Signature | Values |
|---|---|
| Temperature | 80 °C (spike) |
| Pressure | 2 bar |
| Vibration | 50 Hz (peak) |
| Flow | 0 (no material feed) |
| Tool Position | 15 mm (held) |
| Energy | 5.0 kWh (peak) |

**Rationale:** The dramatic **temperature spike combined with peak energy consumption** and zero material flow strongly suggests **welding**. Welding requires intense localized heat, high energy input, and no material movement — the tool is stationary, bonding components in place.

---

###  **Phase 5: Quality Inspection (Events 10–12)**
| Signature | Values |
|---|---|
| Temperature | 30  25 °C (cooling) |
| Pressure | 1 bar (released) |
| Vibration | 1–5 Hz (minimal) |
| Flow | 0  1 units/s |
| Tool Position | 15  20 mm (scanning movement) |
| Energy | 1.0–1.6 kWh (low) |

**Rationale:** Temperature falls as the piece cools; vibration is near zero and energy drops. The small, gentle movement of the tool position (15  20 mm) with very low flow suggests a **sensor/probe scanning** the workpiece — consistent with **quality inspection** rather than active machining.

---

###  **Phase 6: Idle / Shutdown (Events 13–15)**
| Signature | Values |
|---|---|
| Temperature | 20–21 °C |
| Pressure | 1 bar |
| Vibration | 0 Hz |
| Flow / Tool | 0 / 0 (reset) |
| Energy | 0.40–0.50 kWh |

**Rationale:** Readings return to baseline. Tool position resets to 0, indicating the cycle is complete and the machine is **idle / shutdown** awaiting the next job.

---

## Inferred Process Model

```
[Idle/Startup]  [Cutting Metal]  [Assembling Parts]  [Welding] 
      
[Quality Inspection]  [Idle/Shutdown]
```

## Key Heuristics Used

| Indicator | Interpretation |
|---|---|
| High vibration + rising temp + flow | Cutting |
| Moderate steady readings + higher pressure | Assembling |
| Temp spike + energy peak + no flow | Welding |
| Cooling + low vibration + tool scanning | Inspection |
| All sensors at baseline | Idle |

This labeled event log now represents a complete manufacturing cycle mined from raw sensor data — transforming numeric telemetry into an interpretable, auditable process trace.
