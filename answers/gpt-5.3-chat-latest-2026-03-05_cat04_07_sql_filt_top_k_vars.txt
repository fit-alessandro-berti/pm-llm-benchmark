Use an ordered aggregation to build the activity sequence per case, rank variants by frequency, then filter the original log.

Example (replace `K` with the desired number):

```
WITH case_sequences AS (
    SELECT
        case_id,
        string_agg(activity, '->' ORDER BY timestamp) AS variant
    FROM event_log
    GROUP BY case_id
),
variant_counts AS (
    SELECT
        variant,
        COUNT(*) AS case_count
    FROM case_sequences
    GROUP BY variant
),
top_variants AS (
    SELECT variant
    FROM variant_counts
    ORDER BY case_count DESC
    LIMIT K
),
cases_in_top_variants AS (
    SELECT cs.case_id
    FROM case_sequences cs
    JOIN top_variants tv
      ON cs.variant = tv.variant
)
SELECT e.*
FROM event_log e
JOIN cases_in_top_variants c
  ON e.case_id = c.case_id
ORDER BY e.case_id, e.timestamp;
```
