```sql
WITH valid_cases AS (
    SELECT case_id
    FROM event_log
    GROUP BY case_id
    HAVING COUNT(*) <= 6
)
SELECT e.*
FROM event_log AS e
JOIN valid_cases AS vc
    ON e.case_id = vc.case_id
ORDER BY e.case_id, e.timestamp;
```

This query:

- Counts the number of events per `case_id`
- Keeps only cases with `COUNT(*) <= 6`
- Returns all events belonging to those valid cases

A shorter DuckDB version using a window function is:

```sql
SELECT *
FROM event_log
QUALIFY COUNT(*) OVER (PARTITION BY case_id) <= 6
ORDER BY case_id, timestamp;
```
