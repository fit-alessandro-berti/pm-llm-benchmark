Below are the two POWL models based on the described hiring process at GlobalTeq Solutions.

### 1. POWL Model with Potential Unfairness
This model includes the XOR branching logic where candidates are split between a standard cultural fit evaluation and a community affiliation check, which introduces the described bias.

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Define Activities
receive_app = Transition(label="ReceiveApplication")
resume_parsing = Transition(label="ResumeParsing")
request_info = Transition(label="RequestMoreInfo")
skill_assessment = Transition(label="SkillAssessment")
cultural_fit = Transition(label="CulturalFitCheck")
community_check = Transition(label="CommunityAffiliationCheck")
managerial_review = Transition(label="ManagerialReview")
final_decision = Transition(label="FinalDecision")

# 1. Loop for Resume Parsing & Data Completeness
# Applicant parses resume, and if incomplete, requests more info and parses again.
data_loop = OperatorPOWL(operator=Operator.LOOP, children=[resume_parsing, request_info])

# 2. XOR for Cultural Fit vs Community Affiliation (The Bias Point)
# Either the standard check OR the biased affiliation check is performed.
fit_choice = OperatorPOWL(operator=Operator.XOR, children=[cultural_fit, community_check])

# Define the Partial Order
nodes_unfair = [receive_app, data_loop, skill_assessment, fit_choice, managerial_review, final_decision]
model_unfair = StrictPartialOrder(nodes=nodes_unfair)

# Establish sequential order
model_unfair.order.add_edge(receive_app, data_loop)
model_unfair.order.add_edge(data_loop, skill_assessment)
model_unfair.order.add_edge(skill_assessment, fit_choice)
model_unfair.order.add_edge(fit_choice, managerial_review)
model_unfair.order.add_edge(managerial_review, final_decision)
```

---

### 2. POWL Model Without Unfairness
In this model, the XOR branch is removed. Every candidate follows the exact same path, undergoing the standard cultural fit evaluation regardless of their community affiliations.

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Define Activities
receive_app_fair = Transition(label="ReceiveApplication")
resume_parsing_fair = Transition(label="ResumeParsing")
request_info_fair = Transition(label="RequestMoreInfo")
skill_assessment_fair = Transition(label="SkillAssessment")
cultural_fit_fair = Transition(label="CulturalFitCheck")
managerial_review_fair = Transition(label="ManagerialReview")
final_decision_fair = Transition(label="FinalDecision")

# 1. Loop for Resume Parsing & Data Completeness (Maintained for process integrity)
data_loop_fair = OperatorPOWL(operator=Operator.LOOP, children=[resume_parsing_fair, request_info_fair])

# Note: The OperatorPOWL XOR is removed here. 
# All candidates now proceed directly to the CulturalFitCheck.

# Define the Partial Order
nodes_fair = [receive_app_fair, data_loop_fair, skill_assessment_fair, cultural_fit_fair, managerial_review_fair, final_decision_fair]
model_fair = StrictPartialOrder(nodes=nodes_fair)

# Establish sequential order
model_fair.order.add_edge(receive_app_fair, data_loop_fair)
model_fair.order.add_edge(data_loop_fair, skill_assessment_fair)
model_fair.order.add_edge(skill_assessment_fair, cultural_fit_fair)
model_fair.order.add_edge(cultural_fit_fair, managerial_review_fair)
model_fair.order.add_edge(managerial_review_fair, final_decision_fair)
```

### Key Differences Summary:
*   **Control Flow:** The `model_unfair` uses an `OperatorPOWL` with `Operator.XOR` to create two different evaluative paths. The `model_fair` replaces this operator with a single `Transition` (`CulturalFitCheck`).
*   **Bias Mitigation:** By removing the `CommunityAffiliationCheck` branch, the second model ensures that "local resident" or "club membership" flags cannot trigger a separate, subjective scoring path, thereby standardizing the candidate experience.
