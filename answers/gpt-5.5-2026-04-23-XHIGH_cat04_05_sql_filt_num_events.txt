```sql
WITH valid_cases AS (
    SELECT case_id
    FROM event_log
    GROUP BY case_id
    HAVING COUNT(*) <= 6
)
SELECT e.*
FROM event_log AS e
JOIN valid_cases AS v
    ON e.case_id = v.case_id
ORDER BY e.case_id, e.timestamp;
```

This counts events per `case_id`, keeps only cases with six or fewer events, and then returns all events belonging to those valid cases.
