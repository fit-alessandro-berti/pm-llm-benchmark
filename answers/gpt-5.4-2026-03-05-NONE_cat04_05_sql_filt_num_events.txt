```sql
SELECT e.*
FROM event_log AS e
JOIN (
    SELECT case_id
    FROM event_log
    GROUP BY case_id
    HAVING COUNT(*) <= 6
) AS valid_cases
ON e.case_id = valid_cases.case_id
ORDER BY e.case_id, e.timestamp;
```

Alternative using a window function:

```sql
SELECT case_id, activity, timestamp
FROM (
    SELECT *,
           COUNT(*) OVER (PARTITION BY case_id) AS event_count
    FROM event_log
) t
WHERE event_count <= 6
ORDER BY case_id, timestamp;
```

Both queries return all events belonging only to cases with six or fewer total events.
