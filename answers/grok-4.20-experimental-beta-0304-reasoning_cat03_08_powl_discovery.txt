**POWL Model:**

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Activities
OR = Transition(label="OR")
MS = Transition(label="MS")
QS = Transition(label="QS")
AS = Transition(label="AS")
IQC = Transition(label="IQC")
TST = Transition(label="TST")
RT = Transition(label="RT")
PK = Transition(label="PK")
LB = Transition(label="LB")
DP = Transition(label="DP")
D = Transition(label="D")
skip = SilentTransition()

# Loops and choices
assembly_quality_loop = OperatorPOWL(operator=Operator.LOOP, children=[AS, IQC])   # *(AS, IQC)  AS(IQC AS)*
testing_loop = OperatorPOWL(operator=Operator.LOOP, children=[TST, RT])            # *(TST, RT)
labeling_choice = OperatorPOWL(operator=Operator.XOR, children=[LB, skip])        # X(LB, )

# Main partial order (total order in this process)
root = StrictPartialOrder(nodes=[
    OR, 
    MS, 
    QS, 
    assembly_quality_loop, 
    testing_loop, 
    PK, 
    labeling_choice, 
    DP, 
    D
])

root.order.add_edge(OR, MS)
root.order.add_edge(MS, QS)
root.order.add_edge(QS, assembly_quality_loop)
root.order.add_edge(assembly_quality_loop, testing_loop)
root.order.add_edge(testing_loop, PK)
root.order.add_edge(PK, labeling_choice)
root.order.add_edge(labeling_choice, DP)
root.order.add_edge(DP, D)
```

**Explanation of modeling decisions:**

- **Sequential backbone** (`OR  MS  QS  ...  D`) is encoded as a `StrictPartialOrder` with explicit edges. This is the natural POWL way to represent a total order.
- **Assembly & Quality** modeled as `*(AS, IQC)` — captures the repeating "AS  IQC  (AS  IQC ...)" pattern seen in cases 1, 2, 4, and 5. Case 3 (consecutive IQCs) is an outlier but still approximately represented.
- **Testing** modeled as `*(TST, RT)` — perfectly matches all observed behavior (single TST or TST  RT  TST  ...).
- **Labeling** modeled as `X(LB, )` — directly reflects that LB is optional (skipped in case 2).
- All other activities appear in every trace, so they remain mandatory in the partial order.

This POWL model reproduces all observed traces (with minor approximation on the double-IQC in case 3) while using the full expressive power of POWL operators and partial orders.
