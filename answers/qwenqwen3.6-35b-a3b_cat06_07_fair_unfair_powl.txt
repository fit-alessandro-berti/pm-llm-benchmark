### 1. POWL Model with Potential Unfairness
This model introduces the `XOR` branch at the cultural fit stage. One path applies a standard evaluation, while the other path (`CommunityAffiliationCheck`) structurally represents the implicit scoring uplift for local/club-affiliated candidates.

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition
from pm4py.objects.process_tree.obj import Operator

# --- Activities ---
ReceiveApplication = Transition(label="ReceiveApplication")
DataCompletenessCheck = Transition(label="DataCompletenessCheck")
RequestMoreInfo = Transition(label="RequestMoreInfo")
SkillAssessment = Transition(label="SkillAssessment")
CulturalFitCheck = Transition(label="CulturalFitCheck")
CommunityAffiliationCheck = Transition(label="CommunityAffiliationCheck")
ManagerialReview = Transition(label="ManagerialReview")
FinalDecision = Transition(label="FinalDecision")

# --- Operators ---
# Loop: Check data -> if missing, request info -> loop back to check
LoopDataCheck = OperatorPOWL(operator=Operator.LOOP, children=[DataCompletenessCheck, RequestMoreInfo])

# XOR Branch: Introduces the bias point
# Path 1: Standard cultural fit evaluation
# Path 2: Community affiliation check (grants implicit advantage to local/club members)
XORCultureAffiliation = OperatorPOWL(operator=Operator.XOR, children=[CulturalFitCheck, CommunityAffiliationCheck])

# --- Partial Order Construction ---
nodes_unfair = [ReceiveApplication, LoopDataCheck, SkillAssessment, XORCultureAffiliation, ManagerialReview, FinalDecision]
root_unfair = StrictPartialOrder(nodes=nodes_unfair)

# Define sequential execution flow
root_unfair.order.add_edge(ReceiveApplication, LoopDataCheck)
root_unfair.order.add_edge(LoopDataCheck, SkillAssessment)
root_unfair.order.add_edge(SkillAssessment, XORCultureAffiliation)
root_unfair.order.add_edge(XORCultureAffiliation, ManagerialReview)
root_unfair.order.add_edge(ManagerialReview, FinalDecision)
```
