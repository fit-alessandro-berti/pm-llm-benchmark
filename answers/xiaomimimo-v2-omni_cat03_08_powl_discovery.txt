## **POWL Model Definition**

```python
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Define all activity transitions
OR = Transition(label="Order Received")
MS = Transition(label="Material Sourcing")
QS = Transition(label="Quality Screening")
AS = Transition(label="Assembly")
IQC = Transition(label="In-line Quality Check")
TST = Transition(label="Testing")
RT = Transition(label="Re-Testing")
PK = Transition(label="Packaging")
LB = Transition(label="Labeling")
DP = Transition(label="Distribution Planning")
D = Transition(label="Dispatch")
skip = SilentTransition()  # For optional/skipped activities

# 1. Core production sequence (always occurs)
core_sequence = StrictPartialOrder(nodes=[OR, MS, QS, AS])
core_sequence.order.add_edge(OR, MS)
core_sequence.order.add_edge(MS, QS)
core_sequence.order.add_edge(QS, AS)

# 2. Rework loop for quality issues (optional, can repeat)
rework_loop = OperatorPOWL(operator=Operator.LOOP, children=[IQC, AS])

# 3. Testing loop for product validation (optional, can repeat)
testing_loop = OperatorPOWL(operator=Operator.LOOP, children=[RT, TST])

# 4. Labeling choice (can be skipped)
labeling_choice = OperatorPOWL(operator=Operator.XOR, children=[LB, skip])

# 5. Main process structure - sequential with embedded choices and loops
main_process = StrictPartialOrder(nodes=[
    core_sequence,      # Always happens first
    rework_loop,        # Optional quality rework
    TST,                # Initial testing
    testing_loop,       # Optional re-testing loop
    PK,                 # Packaging
    labeling_choice,    # Optional labeling
    DP,                 # Distribution planning
    D                   # Final dispatch
])

# Define the partial order dependencies
main_process.order.add_edge(core_sequence, rework_loop)
main_process.order.add_edge(rework_loop, TST)
main_process.order.add_edge(TST, testing_loop)
main_process.order.add_edge(testing_loop, PK)
main_process.order.add_edge(PK, labeling_choice)
main_process.order.add_edge(labeling_choice, DP)
main_process.order.add_edge(DP, D)

# Final root model
root = main_process
```

## **Model Explanation**

This POWL model captures the complex manufacturing process through these key components:

### **Sequential Flow** (Always occurs):
- `Order Received  Material Sourcing  Quality Screening  Assembly`

### **Optional Rework Loop**:
- After initial assembly, **either** proceed directly to testing **or** enter a quality rework loop
- The rework loop: `In-line Quality Check  [Assembly  In-line Quality Check]*` (can repeat until quality passes)

### **Testing Phase**:
- Initial `Testing` after assembly/rework
- Optional re-testing loop: `Testing  [Re-Testing  Testing]*` (for failed tests)

### **Final Steps**:
- `Packaging`  Optional `Labeling` (can be skipped)  `Distribution Planning  Dispatch`

1. **Choice between straight-through and rework paths** (case 4 vs cases 1,2,3,5)
2. **Nested loops** for both quality rework and testing rework
3. **Optional activities** (labeling can be skipped)
4. **Partial order dependencies** ensuring correct sequencing
5. **Concurrent-independent activities** where applicable
