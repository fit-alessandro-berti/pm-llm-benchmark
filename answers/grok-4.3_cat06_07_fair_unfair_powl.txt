**Unfair POWL Model (with potential bias via XOR branch):**

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition
from pm4py.objects.process_tree.obj import Operator

# Activities
receive = Transition(label="ReceiveApplication")
data_check = Transition(label="DataCompletenessCheck")
request_info = Transition(label="RequestMoreInfo")
skill = Transition(label="SkillAssessment")
cultural_standard = Transition(label="CulturalFitCheck")
community_check = Transition(label="CommunityAffiliationCheck")
manager = Transition(label="ManagerialReview")
final = Transition(label="FinalDecision")

# Loop for data completeness
data_loop = OperatorPOWL(operator=Operator.LOOP, children=[data_check, request_info])

# XOR for cultural fit (unfair branch)
cultural_xor = OperatorPOWL(operator=Operator.XOR, children=[cultural_standard, community_check])

# Root partial order
unfair_model = StrictPartialOrder(nodes=[receive, data_loop, skill, cultural_xor, manager, final])
unfair_model.order.add_edge(receive, data_loop)
unfair_model.order.add_edge(data_loop, skill)
unfair_model.order.add_edge(skill, cultural_xor)
unfair_model.order.add_edge(cultural_xor, manager)
unfair_model.order.add_edge(manager, final)
```

**Fair POWL Model (no selective community branch):**

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition
from pm4py.objects.process_tree.obj import Operator

# Activities
receive = Transition(label="ReceiveApplication")
data_check = Transition(label="DataCompletenessCheck")
request_info = Transition(label="RequestMoreInfo")
skill = Transition(label="SkillAssessment")
cultural = Transition(label="CulturalFitCheck")
manager = Transition(label="ManagerialReview")
final = Transition(label="FinalDecision")

# Loop for data completeness
data_loop = OperatorPOWL(operator=Operator.LOOP, children=[data_check, request_info])

# Root partial order (no XOR, same cultural path for all)
fair_model = StrictPartialOrder(nodes=[receive, data_loop, skill, cultural, manager, final])
fair_model.order.add_edge(receive, data_loop)
fair_model.order.add_edge(data_loop, skill)
fair_model.order.add_edge(skill, cultural)
fair_model.order.add_edge(cultural, manager)
fair_model.order.add_edge(manager, final)
```

These models follow the described structure: `ReceiveApplication`  data completeness loop  `SkillAssessment`  cultural evaluation  `ManagerialReview`  `FinalDecision`. The unfair model uses an explicit XOR after skill assessment to introduce the selective `CommunityAffiliationCheck` path, while the fair model enforces a single `CulturalFitCheck` for all applicants.
