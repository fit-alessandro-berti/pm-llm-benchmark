```text
->(
  'Receive Application (RA)',
  'Validate Form (VF)',
  'Assess Eligibility (AE)',
  X(
    tau,
    ->(
      *(
        'Request Missing Info (RMI)',
        tau
      ),
      'Validate Form (VF)'
    )
  ),
  'Approval Decision (AD)',
  'Send Confirmation (SC)',
  'Archive Case (AC)'
)
```

This represents:

```text
RA -> VF -> AE -> [optional: one or more RMI, then VF] -> AD -> SC -> AC
```

The loop

```text
*( 'Request Missing Info (RMI)', tau )
```

allows one or more repetitions of `RMI`, so it covers both:

```text
RMI -> VF
```

and

```text
RMI -> RMI -> VF
```

while the surrounding XOR with `tau` allows the process to skip the missing-information path entirely.
