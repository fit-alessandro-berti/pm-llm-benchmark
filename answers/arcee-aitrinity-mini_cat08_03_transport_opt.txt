**1. Process Discovery and Conformance Checking:**

**Preprocessing & Integration:**
- **Data Fusion:** Combine event logs using a unified temporal schema. For each vehicle-day (e.g., `V12-20241205`), synchronize GPS, scanner, dispatch, and maintenance logs via timestamps. Tools like ProM or Celonis use *event matching* algorithms to align cross-source events.
- **Challenges:**
  - **Inconsistent Timestamps:** GPS may lag scanners; handle with timestamp alignment techniques (e.g., interpolation).
  - **Missing Data:** E.g., failed scans; impute using historical averages or flag for human review.

  - **Scalability:** For 6 months of data, use incremental processing to avoid memory overload.
  
- **Output:** A unified event log with columns: Timestamp, Case ID (e.g., `V12-20241205`), Activity (e.g., `Depart Depot`, `Delivery Failed`), Resource (Driver ID), Cost Metrics (e.g., `Fuel Consumed`), and annotations.
