```python
activities = [
    "RO",   # Receive Order
    "CA",   # Check Availability
    "RI",   # Reserve Items
    "PI",   # Pick Items
    "QC",   # Quality Check
    "PO",   # Pack Order
    "GSL",  # Generate Shipping Label
    "D",    # Dispatch
    "IC",   # Invoice Customer
    "RP"    # Record Payment
]

log_skeleton = {
    # Since every activity occurs exactly once in the normal process,
    # all activities have equivalent occurrence counts.
    "equivalence": {
        (a, b)
        for a in activities
        for b in activities
        if a != b
    },

    # If an activity occurs, all previous activities in the process
    # must already have occurred.
    "always_before": {
        (activities[i], activities[j])
        for i in range(len(activities))
        for j in range(i)
    },

    # If an activity occurs, all following activities in the process
    # must occur later in the case.
    "always_after": {
        (activities[i], activities[j])
        for i in range(len(activities))
        for j in range(i + 1, len(activities))
    },

    # No pair of activities is mutually exclusive in the described
    # order fulfillment process.
    "never_together": set(),

    # Each activity occurs exactly once per order case.
    "activ_freq": {
        "RO": {1},
        "CA": {1},
        "RI": {1},
        "PI": {1},
        "QC": {1},
        "PO": {1},
        "GSL": {1},
        "D": {1},
        "IC": {1},
        "RP": {1}
    },

    # The normal directly-follows sequence of the fulfillment process.
    "directly_follows": {
        ("RO", "CA"),
        ("CA", "RI"),
        ("RI", "PI"),
        ("PI", "QC"),
        ("QC", "PO"),
        ("PO", "GSL"),
        ("GSL", "D"),
        ("D", "IC"),
        ("IC", "RP")
    }
}
```

This represents the happy-path process:

```text
RO → CA → RI → PI → QC → PO → GSL → D → IC → RP
```
