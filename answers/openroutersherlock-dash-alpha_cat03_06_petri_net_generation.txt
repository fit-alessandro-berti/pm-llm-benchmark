```python
from pm4py.objects.petri_net.obj import PetriNet, Marking
from pm4py.objects.petri_net.utils import petri_utils

# Create the Petri net
net = PetriNet("Insurance_Claim_Management")

# Places representing states between activities
start = PetriNet.Place("start")
p1 = PetriNet.Place("p1")  # after CR
p2 = PetriNet.Place("p2")  # after DV, ready for FC
p3 = PetriNet.Place("p3")  # after FC, ready for CA
p4 = PetriNet.Place("p4")  # after CA, ready for AP decision
p5 = PetriNet.Place("p5")  # after AP, ready for P
p6 = PetriNet.Place("p6")  # after P, ready for N
p7 = PetriNet.Place("p7")  # after N, ready for CL
end = PetriNet.Place("end")

# Re-submission loop places
p_rds = PetriNet.Place("p_rds")  # after RDS, back to FC

# Investigation loop places  
p_ii = PetriNet.Place("p_ii")    # after II, back to FC

# Add all places to the net
net.places.add(start)
net.places.add(p1)
net.places.add(p2)
net.places.add(p3)
net.places.add(p4)
net.places.add(p5)
net.places.add(p6)
net.places.add(p7)
net.places.add(p_rds)
net.places.add(p_ii)
net.places.add(end)

# Transitions with activity labels
CR = PetriNet.Transition("CR", "Claim Registration")
DV = PetriNet.Transition("DV", "Document Verification")
FC = PetriNet.Transition("FC", "Fraud Check")
CA = PetriNet.Transition("CA", "Claim Assessment")
AP = PetriNet.Transition("AP", "Approval")
P = PetriNet.Transition("P", "Payment")
N = PetriNet.Transition("N", "Notification")
CL = PetriNet.Transition("CL", "Closure")

# Re-submission and investigation transitions
RDS = PetriNet.Transition("RDS", "Re-Document Submission")
II = PetriNet.Transition("II", "In-depth Investigation")

# Add all transitions to the net
net.transitions.add(CR)
net.transitions.add(DV)
net.transitions.add(FC)
net.transitions.add(CA)
net.transitions.add(AP)
net.transitions.add(P)
net.transitions.add(N)
net.transitions.add(CL)
net.transitions.add(RDS)
net.transitions.add(II)

# Main process flow
petri_utils.add_arc_from_to(start, CR, net)
petri_utils.add_arc_from_to(CR, p1, net)
petri_utils.add_arc_from_to(p1, DV, net)
petri_utils.add_arc_from_to(DV, p2, net)

# DV -> FC (normal) or DV -> RDS (documents incomplete)
petri_utils.add_arc_from_to(p2, FC, net)
petri_utils.add_arc_from_to(p2, RDS, net)

# RDS -> FC (documents resubmitted)
petri_utils.add_arc_from_to(RDS, p_rds, net)
petri_utils.add_arc_from_to(p_rds, FC, net)

# FC -> CA (normal) or FC -> II (fraud doubts)
petri_utils.add_arc_from_to(FC, p3, net)
petri_utils.add_arc_from_to(p3, CA, net)
petri_utils.add_arc_from_to(p3, II, net)

# II -> FC (loop back for re-check, can happen multiple times)
petri_utils.add_arc_from_to(II, p_ii, net)
petri_utils.add_arc_from_to(p_ii, FC, net)

# Continue main flow
petri_utils.add_arc_from_to(CA, p4, net)
petri_utils.add_arc_from_to(p4, AP, net)
petri_utils.add_arc_from_to(AP, p5, net)
petri_utils.add_arc_from_to(p5, P, net)
petri_utils.add_arc_from_to(P, p6, net)
petri_utils.add_arc_from_to(p6, N, net)
petri_utils.add_arc_from_to(N, p7, net)
petri_utils.add_arc_from_to(p7, CL, net)
petri_utils.add_arc_from_to(CL, end, net)

# Initial and final markings
im = Marking()
im[start] = 1

fm = Marking()
fm[end] = 1
```

This accepting Petri net models the complete insurance claim management process with:

1. **Main sequential flow**: C  CR  DV  FC  CA  AP  P  N  CL
2. **Document re-submission loop**: After DV, can go to RDS and back to FC
3. **Fraud investigation loop**: After FC, can go to II and back to FC (multiple times possible)
4. **Proper place-token management** with start place having 1 token initially and end place as final marking
5. **All transitions properly labeled** with activity names and connected via arcs

The net correctly handles the loops while ensuring every process reaches the final "end" state after closure.
