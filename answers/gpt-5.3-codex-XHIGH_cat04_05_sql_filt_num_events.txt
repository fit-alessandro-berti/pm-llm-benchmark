You can do this cleanly with a `GROUP BY` + `HAVING` in a CTE, then join back to get all events for valid cases:

```sql
WITH eligible_cases AS (
    SELECT case_id
    FROM event_log
    GROUP BY case_id
    HAVING COUNT(*) <= 6
)
SELECT e.*
FROM event_log AS e
JOIN eligible_cases AS c
  ON e.case_id = c.case_id
ORDER BY e.case_id, e."timestamp";
```

This counts events per `case_id`, keeps only cases with 6 or fewer events, and returns all event rows for those cases.
