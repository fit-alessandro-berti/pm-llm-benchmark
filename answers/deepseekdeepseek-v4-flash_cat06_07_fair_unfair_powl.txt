Below are two POWL models constructed from the described hiring process. The first model includes an XOR branching that can introduce unfairness through a communityaffiliation check; the second model removes that branch and treats all applicants uniformly.

## 1. POWL Model with Potential Unfairness

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Activities (transitions)
ReceiveApp = Transition(label="ReceiveApplication")
DataCheck  = Transition(label="DataCompletenessCheck")
RequestMore = Transition(label="RequestMoreInfo")
SkillAssess = Transition(label="SkillAssessment")
CulturalFit = Transition(label="CulturalFitCheck")
CommunityCheck = Transition(label="CommunityAffiliationCheck")
MgmtReview = Transition(label="ManagerialReview")
FinalDec   = Transition(label="FinalDecision")

# Loop: DataCompletenessCheck -> (if incomplete) RequestMoreInfo -> back to DataCompletenessCheck
loop_data = OperatorPOWL(
    operator=Operator.LOOP,
    children=[DataCheck, RequestMore]
)

# XOR: either standard CulturalFitCheck or CommunityAffiliationCheck
xor_affiliation = OperatorPOWL(
    operator=Operator.XOR,
    children=[CulturalFit, CommunityCheck]
)

# Partial order: sequential flow
root_unfair = StrictPartialOrder(nodes=[
    ReceiveApp,
    loop_data,
    SkillAssess,
    xor_affiliation,
    MgmtReview,
    FinalDec
])
root_unfair.order.add_edge(ReceiveApp, loop_data)
root_unfair.order.add_edge(loop_data, SkillAssess)
root_unfair.order.add_edge(SkillAssess, xor_affiliation)
root_unfair.order.add_edge(xor_affiliation, MgmtReview)
root_unfair.order.add_edge(MgmtReview, FinalDec)

# The model is now stored in root_unfair
# To visualize: pm4py.view_powl(root_unfair)
```

**Explanation of the unfair model:**
- After `SkillAssessment`, an XOR operator splits the flow.
- One branch executes a standard `CulturalFitCheck`.
- The other branch executes a `CommunityAffiliationCheck`, which in the textual description gives a subtle advantage to applicants with local community ties.
- This selective bias is the source of unfairness.
