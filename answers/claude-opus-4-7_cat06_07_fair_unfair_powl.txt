Here are the two POWL models representing the hiring process at GlobalTeq Solutions:

## Model 1: POWL Model with Potential Unfairness

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Define activities
receive_application = Transition(label="ReceiveApplication")
data_completeness_check = Transition(label="DataCompletenessCheck")
request_more_info = Transition(label="RequestMoreInfo")
skill_assessment = Transition(label="SkillAssessment")
cultural_fit_check = Transition(label="CulturalFitCheck")
community_affiliation_check = Transition(label="CommunityAffiliationCheck")
managerial_review = Transition(label="ManagerialReview")
final_decision = Transition(label="FinalDecision")

# Loop: verify data completeness; if incomplete, request more info and re-check
data_loop = OperatorPOWL(operator=Operator.LOOP, 
                        children=[data_completeness_check, request_more_info])

# XOR: standard cultural fit OR community affiliation check (source of bias)
biased_xor = OperatorPOWL(operator=Operator.XOR, 
                          children=[cultural_fit_check, community_affiliation_check])

# Build partial order representing the full sequential flow
root_unfair = StrictPartialOrder(nodes=[
    receive_application, 
    data_loop, 
    skill_assessment, 
    biased_xor, 
    managerial_review, 
    final_decision
])
root_unfair.order.add_edge(receive_application, data_loop)
root_unfair.order.add_edge(data_loop, skill_assessment)
root_unfair.order.add_edge(skill_assessment, biased_xor)
root_unfair.order.add_edge(biased_xor, managerial_review)
root_unfair.order.add_edge(managerial_review, final_decision)
```

**Where unfairness appears:** The `biased_xor` node offers two alternative evaluation paths. One path is the standard `CulturalFitCheck`, while the other is a `CommunityAffiliationCheck` that grants an implicit uplift to applicants with local community ties or membership in the "Global Business Leaders Club." Because the choice of path is not merit-based, the XOR branch introduces subtle bias that can tilt the `ManagerialReview` and `FinalDecision` in favor of affiliated applicants.

---

## Model 2: POWL Model Without Unfairness

```python
import pm4py
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# Define activities
receive_application_f = Transition(label="ReceiveApplication")
data_completeness_check_f = Transition(label="DataCompletenessCheck")
request_more_info_f = Transition(label="RequestMoreInfo")
skill_assessment_f = Transition(label="SkillAssessment")
cultural_fit_check_f = Transition(label="CulturalFitCheck")
managerial_review_f = Transition(label="ManagerialReview")
final_decision_f = Transition(label="FinalDecision")

# Loop: verify data completeness; if incomplete, request more info and re-check
data_loop_f = OperatorPOWL(operator=Operator.LOOP, 
                          children=[data_completeness_check_f, request_more_info_f])

# No XOR branch: every applicant goes through the same CulturalFitCheck
root_fair = StrictPartialOrder(nodes=[
    receive_application_f, 
    data_loop_f, 
    skill_assessment_f, 
    cultural_fit_check_f, 
    managerial_review_f, 
    final_decision_f
])
root_fair.order.add_edge(receive_application_f, data_loop_f)
root_fair.order.add_edge(data_loop_f, skill_assessment_f)
root_fair.order.add_edge(skill_assessment_f, cultural_fit_check_f)
root_fair.order.add_edge(cultural_fit_check_f, managerial_review_f)
root_fair.order.add_edge(managerial_review_f, final_decision_f)
```

**Why this model is fair:** The XOR branch has been eliminated. There is no `CommunityAffiliationCheck` activity, and every applicant is evaluated solely through the standard `CulturalFitCheck`. The loop for data completeness is preserved (it is a neutral data-quality mechanism, not a discriminatory branch), and the sequential order of screening, skill assessment, cultural evaluation, managerial review, and final decision is retained. By removing the selective affiliation-based path, the process ensures that all candidates are assessed under the same criteria, mitigating the implicit bias source.
