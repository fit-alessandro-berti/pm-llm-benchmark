Here are the two POWL models implemented in Python using `pm4py`. Both models capture the hiring process described, with the first model including the XOR branch that introduces potential bias, and the second model removing that branch to ensure fairness.

### 1. POWL Model with Potential Unfairness

This model includes the **XOR branch** after the skill assessment. One path performs a standard cultural fit evaluation, while the other performs a `CommunityAffiliationCheck` that can introduce subtle bias based on local affiliations.

```python
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# --- Activities ---
receive_app = Transition(label="ReceiveApplication")
data_check  = Transition(label="DataCompletenessCheck")
req_info    = Transition(label="RequestMoreInfo")
skill_assess = Transition(label="SkillAssessment")
cultural_fit = Transition(label="CulturalFitCheck")
comm_affil  = Transition(label="CommunityAffiliationCheck")  # Bias-prone path
mgr_review  = Transition(label="ManagerialReview")
final_dec   = Transition(label="FinalDecision")

# --- Structure ---
# Loop: Check data completeness; if missing, request more info and re-check
loop_data = OperatorPOWL(operator=Operator.LOOP, children=[data_check, req_info])

# XOR: Standard cultural fit OR community-affiliation-based check (potential bias)
xor_culture = OperatorPOWL(operator=Operator.XOR, children=[cultural_fit, comm_affil])

# Build partial order
nodes_m1 = [receive_app, loop_data, skill_assess, xor_culture, mgr_review, final_dec]
po_m1 = StrictPartialOrder(nodes=nodes_m1)

# Define execution order
po_m1.order.add_edge(receive_app, loop_data)
po_m1.order.add_edge(loop_data, skill_assess)
po_m1.order.add_edge(skill_assess, xor_culture)
po_m1.order.add_edge(xor_culture, mgr_review)
po_m1.order.add_edge(mgr_review, final_dec)

# po_m1 is now the POWL model for the biased hiring process
```

### 2. POWL Model Without Unfairness

This model ensures **all applicants** undergo the same cultural fit evaluation. The XOR branch and community-affiliation check are removed, resulting in a uniform, bias-free workflow while preserving the loop for data completeness and the sequential assessment steps.

```python
from pm4py.objects.powl.obj import StrictPartialOrder, OperatorPOWL, Transition, SilentTransition
from pm4py.objects.process_tree.obj import Operator

# --- Activities ---
receive_app = Transition(label="ReceiveApplication")
data_check  = Transition(label="DataCompletenessCheck")
req_info    = Transition(label="RequestMoreInfo")
skill_assess = Transition(label="SkillAssessment")
cultural_fit = Transition(label="CulturalFitCheck")  # Standard path only
mgr_review  = Transition(label="ManagerialReview")
final_dec   = Transition(label="FinalDecision")

# --- Structure ---
# Loop: Check data completeness; if missing, request more info and re-check
loop_data = OperatorPOWL(operator=Operator.LOOP, children=[data_check, req_info])

# No XOR branch; all candidates follow the same cultural fit path
# Build partial order
nodes_m2 = [receive_app, loop_data, skill_assess, cultural_fit, mgr_review, final_dec]
po_m2 = StrictPartialOrder(nodes=nodes_m2)

# Define execution order
po_m2.order.add_edge(receive_app, loop_data)
po_m2.order.add_edge(loop_data, skill_assess)
po_m2.order.add_edge(skill_assess, cultural_fit)
po_m2.order.add_edge(cultural_fit, mgr_review)
po_m2.order.add_edge(mgr_review, final_dec)

# po_m2 is now the POWL model for the fair hiring process
```

### Key Differences
- **Model 1** uses `OperatorPOWL(operator=Operator.XOR, ...)` to introduce a branching path where applicants may be routed to a `CommunityAffiliationCheck`, enabling the subtle bias described in the text.
- **Model 2** replaces the XOR branch with a direct edge to `CulturalFitCheck`, ensuring every applicant is evaluated uniformly without affiliation-based adjustments.
- Both models preserve the `LOOP` structure for data completeness and the sequential flow of skill assessment and managerial review.
