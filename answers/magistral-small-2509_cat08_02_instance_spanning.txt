### 1. Identifying Instance-Spanning Constraints and Their Impact

To formally identify and quantify the impact of instance-spanning constraints, we would employ process mining techniques on the event log. For each constraint, we would derive specific metrics to measure their impact.

**Cold-Packing Stations:**
- **Metric:** Average waiting time for orders requiring cold-packing at the Packing step.
- **Analysis:** By comparing the timestamp when an order is ready for Packing (after Item Picking completes) with the timestamp when it actually starts Packing, we can calculate the waiting time. High waiting times indicate resource contention.

**Shipping Batches:**
- **Metric:** Batch waiting time, defined as the time from the last order in a batch completing Quality Check to the batch's Shipping Label Generation start.
- **Analysis:** This measures how long orders wait for batch completion. High values suggest inefficiencies in batch formation.

**Priority Handling:**
- **Metric:** Average delay to standard orders when an express order preempts a resource.
- **Analysis:** For each instance where an express order preempts a standard order (evident from the log if a standard order's activity is interrupted), calculate the delay incurred.

**Hazardous Materials:**
- **Metric:** Frequency of reaching the 10-order limit and the resulting throughput reduction.
- **Analysis:** Track the number of concurrent hazardous material orders in Packing or Quality Check. Delays occur when the 10th order starts and others must wait.

To differentiate within-instance from between-instance waiting times, we would:
- **Within-instance:** Long activity durations without resource contention (e.g., an order takes longer in Packing due to complexity).
- **Between-instance:** Orders waiting for resources occupied by others (e.g., an order waits for a Cold-Packing station because it's busy with another order).

Constraints interact in complex ways. For example:

Understanding these interactions is crucial because optimizing one constraint in isolation may worsen others. For instance, prioritizing express orders at Cold-Packing stations could increase batching delays if it causes a backlog.
